# Study Plan Expo App

Native Expo version of the interview study-plan app.

This project is pinned to Expo SDK 54 so it runs in Expo Go 54.

## Run

```sh
npm install
npm run start
```

Open the Expo Go app on your phone and scan the QR code from the terminal.

Useful scripts:

```sh
npm run ios
npm run android
npm run web
npm run export:web
npm run lint
npm run doctor
```

## Host On GitHub Pages

Push this project to a GitHub repository, then enable GitHub Pages with
`Settings -> Pages -> Source -> GitHub Actions`.

Every push to `main` will build and publish the web app.
